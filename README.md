# 📊 Real Mart Sales Analysis Dashboard

## 📌 Project Overview
An interactive sales analysis dashboard built using Microsoft Excel, analyzing 3 months of retail data across multiple dimensions including cities, branches, and product lines.

## 📈 Key Metrics
- 💰 Total Sales: ₹659.3K
- 🛒 Total Orders: 2,064
- 📦 Gross Income: ₹31.4K
- ⭐ Avg Customer Rating: 7.2

## 💡 Key Insights
- Branch C was the top performer at ₹347.8K
- Mumbai led city-wise sales at ₹323.7K
- Cash was the dominant payment method at 60%
- Fashion accessories had the highest avg rating at 7.3
- Sales dipped in February but recovered in March

## 📊 Dashboard Features
- ✅ KPI Summary Cards
- ✅ Sales by Product Line
- ✅ Sales by City & Branch
- ✅ Payment Method Breakdown
- ✅ Monthly Sales Trend
- ✅ Customer Type Distribution
- ✅ Interactive Slicers & Filters
